# My Dotfiles 🐚

Scripts for quick installation of Zsh and Oh My Zsh.

## How to use
1. Clone repo:
   `git clone https://github.com/sasas991/dotfiles.git`
2. Move to directory:
   `cd dotfiles`
3. Launch script:
   `bash install.sh`

## Content
* Installation: Zsh + Oh My Zsh
* Plugins: zsh-autosuggestions, git
* Custom aliases
